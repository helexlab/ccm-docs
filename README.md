# CCM documentation

This project contains user and installation manuals for Care Coordination Management project

## Working with project

1. Install Ruby and Jekyll in the your computer.
2. Navigate to the docs directory.

```bash
cd docs
```

3. Install Ruby gems.

```bash
bundle install
```

4. Run local web server.

```bash
bundle exec jekyll serve --config _config.yml,_config_local.yml --baseurl ""
```

5. Open website on the [http://127.0.0.1:4000](http://127.0.0.1:4000).
